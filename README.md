## Installation

<br>

Clone the repository :

```bash
git clone https://github.com/ksyhoo/cm-rekru
```

<br>

Install dependencies using Yarn or NPM or PNPM :

```bash

# using yarn
yarn install

# or using npm
npm install
```

<br />

## Start : Development

To develop and run your web application, you need to run following command :

```bash
yarn start
```
